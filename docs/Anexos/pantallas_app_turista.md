# Diseño de Pantallas para la App Móvil (Turista)

Este documento describe cómo deberían quedar las pantallas principales de la aplicación móvil para el **Turista**, excluyendo el flujo de creación y login de cuenta.

---

## ✅ Pantallas principales

### 1. Pantalla Principal (Home)
**Elementos:**
- Barra superior con logo y acceso al perfil.
- Tarjetas principales:
  - Mi Viaje Actual (nombre, fechas, guía asignado).
  - Itinerario.
  - Mapa Interactivo.
  - Chat Grupal.
  - Walkie-Talkie.
- Alertas recientes.

**Diseño sugerido:**
- Estilo limpio, tarjetas grandes, botones claros.
- Colores adaptables (tema claro/oscuro).

---

### 2. Pantalla de Itinerario
**Elementos:**
- Lista cronológica de eventos.
- Cada evento muestra hora, lugar, descripción breve.
- Botón para ver mapa.

**Extras:**
- Alertas automáticas para eventos próximos.

---

### 3. Pantalla de Mapa Interactivo
**Elementos:**
- Mapa con ubicación actual del turista y guía.
- Puntos de reunión y eventos próximos.
- Botón para centrar en ubicación actual.

**Extras:**
- Alerta si el turista se aleja demasiado del punto de referencia.

---

### 4. Pantalla de Comunicación
**Elementos:**
- Chat grupal con mensajes en tiempo real.
- Campo para escribir mensaje y botón enviar.
- Walkie-Talkie con botón destacado.

**Extras:**
- Notificaciones en tiempo real.

---

### 5. Pantalla de Configuración
**Elementos:**
- Idioma.
- Tema (claro, oscuro, automático).
- Alertas (activar/desactivar).
- Accesibilidad (tamaño de texto, contraste).

**Extras:**
- Cambios persistentes.

---

### 6. Pantalla de Perfil
**Elementos:**
- Foto.
- Nombre completo, correo, teléfono.
- Botón Editar perfil.
- Botón Eliminar cuenta (con confirmación).

**Extras:**
- Mostrar viajes anteriores.

---
